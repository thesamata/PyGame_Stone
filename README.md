# 🪨 PyGame Stone

Pygame ile geliştirilmiş nostaljik sanal bebek oyunu.

## 🚀 Hızlı Başlangıç
1. `pip install -r requirements.txt`
2. `python main.py`

## 🎮 Kontroller
- **R:** Oyunu başlatır.
- **Besle / Uyku:** İhtiyaçları giderir.
- **Tıklama:** Karakteri sever ve altın kazandırır.
